WordPress User Registration CVE-2026-1492 Checker

A read-only local checker for WordPress owners. It looks for plugin exposure, hidden administrators, PHP files in uploads, cron anomalies, and file changes that need review.

Open repository Read the Ping7 guide Request repair help
Run this only on WordPress sites you own or support. The checker does not modify files and does not provide exploit steps.

What it checks

  • User Registration plugin version and presence.
  • Visible and hidden administrator accounts.
  • PHP-like files under wp-content/uploads.
  • Cron, theme, config, and core-folder anomalies.

Safe run path

Run from the WordPress root where wp-config.php exists.

curl -fsSLO https://raw.githubusercontent.com/limo57640-crypto/wp-user-registration-vuln-checker/main/check.sh
less check.sh
bash check.sh

When to escalate

  • The result is SUSPICIOUS or COMPROMISED.
  • Unknown admins, redirects, upload PHP files, or cron changes appear.
  • The site stores customer, membership, or payment data.
